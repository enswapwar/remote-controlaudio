# remote-controlaudio
これは文字通り、遠隔で音声を操作できるもの。

スピーカーが無く、スマホなどがある場合のスピーカー代わりとして使えたり学校のいたずらなどに使える。
## How to use?
まずrenderなどの動的サーバーで動かす、静的サーバーでは動かない。
build commandは 
```build
npm install
```
を使用し、deployは
```deploy
node server.js
```
を使用し、サバを起動する。

サバが起動できたらindex.htmlに行くと子機の設定に行く、こちらでは子機の名前、音声、そして決定ができる。

名前を入力して、その後音声を選択する。選択ができたら右にある確定を押す。下の未登録が登録済みになったら成功。子機の設定はこれで終わり

親機はadmin.htmlに行く。こちらでは子機の名前を選択し、子機から音を無らしたり、止めたり、音量を調節したり、親機から音声を送信し、その音声を、子機で鳴らしたりできる。

左の欄では子機の名前を選択する（子機で登録済みとなっていなければ表示されない）Play Stopは子機で設定した音を流したり止めたりする。
